Switch #

Switch is used for switching between two opposing states.

Basic usage #

Bind 'v-model' to a variable of type 'Boolean'.

<>
<template>
    <kl-switch v-model="value1"></kl-switch>
</template>

<script setup lang="ts">
import { ref } from 'vue'

const value1 = ref(true)
const value2 = ref(false)
</script>
